EdgeLoopIterator

class EdgeLoopIterator
implements Iterator

Iterator for traversing over the edges forming a loop around a face. It loops in the counterclockwise direction.

In order to guarantee that each edge is encountered only once, the face's edge list not be modified.

You can get the shared iterator of a Face: face[Symbol.iterator]

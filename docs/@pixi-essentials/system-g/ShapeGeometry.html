ShapeGeometry

class ShapeGeometry extends BoundedBufferGeometry

ShapeGeometry is a semantic class that represents a generic shape geometry with the following attribute(s):

  • (vec2) aVertexPosition: the local-space coordinates of the geometry vertices
  • (optional) (vec2) aTextureCoord: the mapped texture coordinates for each vertex

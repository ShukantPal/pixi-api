G

class G

Plugin class for renderer.plugins.g. This API provides utilties for managing PixiJS geometries and their data. This includes:

  • Comparing & merging geometries
  • Manipulating data based on the data-type
  • Factory for typed-array buffers.

Parameters:
Name Type Attributes Default Description
into PIXI.Geometry
geom PIXI.Geometry
startVertex number

<optional>

 0

the vertex at which to start writing "geom" into "into"'s buffer. By default, this is zero and will overwrite "geom".
swap boolean

<optional>

 false

whether the geometry's buffers can be swapped with a larger capacity. If false, then an error will be thrown if "into" does not contain enough capacity.
validateStyle boolean

<optional>

 false

validates whether the style of "geom" is the same as "into". If not, then no modifications are done and it is returend as-is.

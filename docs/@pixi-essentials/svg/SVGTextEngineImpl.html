SVGTextEngineImpl

class SVGTextEngineImpl extends PIXI.Sprite
implements SVGTextEngine

SVGTextEngineImpl is the default implementation for SVGTextEngine. It is inspired by PIXI.Text that is provided by @pixi/text. It uses a <canvas /> to draw and cache the text. This may cause blurring issues when the SVG is viewed at highly zoomed-in scales because it is rasterized.

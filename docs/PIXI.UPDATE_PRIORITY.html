@pixi/ticker

PIXI.UPDATE_PRIORITY

static UPDATE_PRIORITY

Represents the update priorities used by internal PIXI classes when registered with the PIXI.Ticker.html object. Higher priority items are updated first and lower priority items, such as render, should go later.

Public Properties

HIGH

static HIGH: number

High priority updating, PIXI.VideoBaseTexture and {@link PIXI.AnimatedSprite}

INTERACTION

static INTERACTION: number

Highest priority, used for PIXI.InteractionManager.html

LOW

static LOW: number

Low priority used for PIXI.Application.html rendering.

NORMAL

static NORMAL: number

Default priority for ticker events, see PIXI.Ticker#add.

UTILITY

static UTILITY: number

Lowest priority used for PIXI.BasePrepare.html utility.

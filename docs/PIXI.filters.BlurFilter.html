PIXI.filters.BlurFilter

class BlurFilter extends PIXI.Filter

The BlurFilter applies a Gaussian blur to an object.

The strength of the blur can be set for the x-axis and y-axis separately.

If enabled, PixiJS will fit the filter area into boundaries for better performance. Switch it off if it does not work for specific shader.

enabled

enabled: boolean

If enabled is true the filter is applied, if false it will not.

legacy

legacy: boolean

Legacy filters use position and uvs from attributes

padding

padding: number

The padding of the filter. Some filters require extra space to breath such as a blur. Increasing this will add extra width and height to the bounds of the object that the filter is applied to.

program

program: PIXI.Program.html

Program that the shader uses

quality

quality: number

Sets the number of passes for blur. More passes means higher quaility bluring.

repeatEdgePixels

repeatEdgePixels: boolean

If set to true the edge of the target will be clamped

resolution

resolution: number

The resolution of the filter. Setting this to be lower will lower the quality but increase the performance of the filter.

state

state: PIXI.State.html

The WebGL state the filter requires to render

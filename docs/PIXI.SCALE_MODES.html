PIXI.SCALE_MODES

static SCALE_MODES

The scale modes that are supported by pixi.

The PIXI.settings.html#SCALE_MODE scale mode affects the default scaling mode of future operations. It can be re-assigned to either LINEAR or NEAREST, depending upon suitability.

Smooth scaling

SVGTextEngine

interface SVGTextEngine

The SVGTextEngine interface is used to layout text content authored in SVG files. The @pixi-essentials/svg package provides SVGTextEngineImpl.html as a default implementation for users.

Text engines are allowed to have async behaviour so that fonts can be loaded before text metrics are measured.

It is expected an implementation inherits from PIXI.DisplayObject.html.

Public Methods

clear

clear() → {Promise}

Clears the text content laid out already. This should reset the state of the engine to before any calls to SVGTextEngine.html#put were made.

put

put(id: any, position: IPointData, content: string, style: Partial, matrix: PIXI.Matrix) → {Promise}

Puts the text {@code content} into the local space of the engine at {@code position}. {@code matrix} can be used to transform the glyphs, although it is as optional feature for implementations.
